Step 1: The Physics of Tandem Flight

This bicopter utilizes two rotors. Control is achieved by modulating the RPM difference for Pitch and the Servo Angle for Roll/Yaw.

Step 2: Mathematical Foundation

A. Pitch (Differential Thrust)

$$Motor_{Front} = Base - Output_{Pitch}$$ $$Motor_{Back} = Base + Output_{Pitch}$$

B. Roll & Yaw (Vector Mixing)

$$Servo_{Front} = Center + Output_{Roll} + Output_{Yaw}$$ $$Servo_{Back} = Center - Output_{Roll} + Output_{Yaw}$$

Step 3: Flight Controller Core Logic (C++)

The stabilization loop processes MPU6050 data and calculates actuator signals at 100Hz+.


void loop() {
    mpu.update(); // 1. Get IMU Angles
    float roll_act = mpu.getAngleX();
    float pitch_act = mpu.getAngleY();

    // 2. PID Calculations
    float out_r = pidRoll.compute(roll_set - roll_act);
    float out_p = pidPitch.compute(pitch_set - pitch_act);
    float out_y = pidYaw.compute(yaw_set - gyro_z);

    // 3. Actuator Mixing Logic
    int mot_f = throttle - out_p;
    int mot_b = throttle + out_p;
    int srv_f = FRONT_MID + out_r + out_y;
    int srv_b = BACK_MID  - out_r + out_y;

    // 4. Send to PCA9685 Hardware
    if (is_armed && throttle > 1050) {
        pwm.setPWM(ESC_F, 0, constrain(mot_f, 1000, 2000));
        pwm.setPWM(ESC_B, 0, constrain(mot_b, 1000, 2000));
        pwm.setPWM(SRV_F, 0, constrain(srv_f, 150, 450));
        pwm.setPWM(SRV_B, 0, constrain(srv_b, 150, 450));
    }
}

Step 4: Operational Procedures

1. Calibration: Place on a level surface and click 'Calibrate Accelerometer' to zero the IMU.

2. Tuning: Increase P until oscillation occurs, then dampen with D. Use the Blackbox tab to visualize the Step Response.
